دليل إعداد لوحة التحكمNO-CODE ADMIN PANEL SETUP

لوحة تحكم تتيح لكِ إضافة الصور والنصوص بدون أي كود — بعد إعداد لمرة واحدة.

؟ ما هذه اللوحة؟

لوحة التحكم (Decap CMS) صفحة تسجّلين فيها الدخول، ثم تعدّلين كل محتوى الموقع — الإعلانات، الطلاب المتفوقين، الصور، الأسئلة — من خلال حقول جاهزة تكتبين فيها، وترفعين الصور بالسحب. لا حاجة لأي كود.

مهم: هذه اللوحة تحتاج إعداداً لمرة واحدة (الخطوات أدناه). بعدها تصبح جاهزة للاستخدام اليومي البسيط.

1 ارفعي الموقع على GitHub

1أنشئي حساباً مجانياً على github.com إن لم يكن لديكِ.
2أنشئي مستودعاً (Repository) جديداً وارفعي فيه كل ملفات الموقع.
3فعّلي GitHub Pages من إعدادات المستودع (Settings ▸ Pages) ليصبح الموقع أونلاين.

2 اربطي الموقع بـ Netlify (مجاني)

1أنشئي حساباً مجانياً على netlify.com (سجّلي الدخول بحساب GitHub).
2اختاري Add new site ▸ Import an existing project، واختاري مستودع الموقع.
3اضغطي Deploy — سيعطيكِ Netlify رابطاً للموقع.

3 فعّلي تسجيل الدخول للوحة

1في Netlify: Site settings ▸ Identity ▸ Enable Identity.
2فعّلي Git Gateway (تحت Identity ▸ Services).
3تحت Identity ▸ Registration، اختاري Invite only (لمنع تسجيل الغرباء).
4اضغطي Invite users وأدخلي بريدكِ الإلكتروني — ستصلكِ دعوة لتعيين كلمة مرور.

4 عدّلي اسم المستودع في الإعداد

1افتحي ملف admin/config.yml.
2ابحثي عن السطر الذي فيه اسم المستودع وغيّريه لاسم مستودعكِ (لو استخدمتِ Git Gateway مع Netlify فالإعداد الحالي يعمل مباشرة).

✓ الاستخدام اليومي

1ادخلي على: رابط-موقعك/admin
2سجّلي الدخول بكلمة المرور التي عيّنتِها.
3اختاري القسم (إعلانات، طلاب، صور...)، اكتبي أو ارفعي الصورة، واضغطي Publish.
4خلال دقيقة، يتحدّث الموقع تلقائياً. ✅
هذا كل شيء! بعد الإعداد لمرة واحدة، تديرين الموقع بالكامل من لوحة أنيقة — بدون كود ولا رفع يدوي.

i ملاحظة أمينة

ربط لوحة التحكم بالموقع الحالي (بحيث تظهر التعديلات فوراً) يحتاج خطوة تقنية إضافية: تعديل الموقع ليقرأ المحتوى من ملفات اللوحة. أخبريني عندما تكونين جاهزة وقد أنجزتِ الإعداد أعلاه، وسأكمل هذا الربط لكِ.
